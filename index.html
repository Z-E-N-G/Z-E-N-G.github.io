概论

Web前端开发概述

  • Web Web系统：以网站形式呈现，通过浏览器访问，完成一定功能的系统
  • 前端 网页上为用户呈现的部分
  • 开发 编写代码

网站与网页

  • 网站（web site） 互联网上用于展现特定内容的网页的集合→文件夹 网页（web page） 网站中一页，一个网站中的网页通过超链接被组织在一起→文件

  • 主页（homepage） 文件名通常为index

  • 网页元素 表单、Logo站标、导航栏、文字or图片超链接、广告横幅

  • 浏览器 解析网页源代码，渲染网页

    • relative 相对定位，相对于其直接父元素进行定位

    • 只有 static 对于上述属性无效，其余都有效
